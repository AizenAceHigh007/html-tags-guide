Basic HTML Tags

Tag Name Description
<html> HTML The root element of an HTML document.
<head> Head Contains meta-information about the document.
<title> Title Sets the title of the document, shown in the browser's title bar of tab.
<body> Body Contains the content of an HTML document, including text, images, links, etc.

Text Formatting Tags

Tag Name Description
<p> Paragraph Defines a paragraph.
<br> Line Break Inserts a single line break.
<hr> Horizontal Rule Represents a thematic break or a horizontal rule.
<b< Bold Makes the text bold.
<i> Italic Makes the text italic.
<u> Underline Underlines the text.
<strong> Strong Indicates strong importance, usually rendered in bold.
<em> Emphasis Indicates emphasized text, usually rendered in italic.
<small> Small Renders text in a smaller font size.
<mark> Mark Highlights text.
<del> Deleted Text Indicates text that has been deleted.
<ins> Inserted Text Indicates text that has been inserted.
<sub> Subscript Defines subscripted text.
<sup> Superscript Defines superscripted text.
<blockquote> Blockquote Defines a section that is quoted from another source.
<q> Quote Defines a short inline quotation.
<code> Code Defines a fragment of computer code.
<pre> Preformatted Defines preformatted text, typically displayed in a monospace font.
<abbr> Abbreviation Represents an abbreviation or acronym.
<citation> Citation Cites a title of a work.
<dfn> Definition Represents the defining instance of a term.

Semantic Tags

Tag Name Description
<header> Header Represents introductory content, typically a group of introductory or navigational aids.
<footer> Footer Represents the footer for a section or document.
<main> Main Spcifies the main content of a document.
<section> Section Defines a section in a document.
<article> Article Represents an independent piece of content of a document, such as a blog post.
<aside> Aside Defines content aside from the main content, like a sidebar.
<nav> Navigation Defines a set of navigation links.
<figure> Figure Specifies self-contained content, such as illustrations, diagrams, photos, etc.
<figcaption> Figcaption Defines a caption for a <figure> element.

List Tags

Tag Name Description
<ul> Unordered List Defines a unordered (bulleted) list.
<ol> Ordered List Defines an ordered (numbered) list.
<li> List Item Defines a list item.
<dl> Description List Defines a description list.
<dt> Description Term Represents a term in a description list.
<dd> Description Definition Describes a term in a description list.

Form Tags

Tag Name Description
<form> Form Represents a document section containing interactive controls for submitting data.
<input> Input Defines an input control.
<textarea> Textarea Defines a multiline input control (text area).
<button> Button Defines a clickable button.
<select> Select Defines a drop-down list.
<option> Option Defines an option in a drop-down list.
<optgroup> Option Group Groups options in a drop-down list.
<label> Label Defines a label for an <input> element.
<fieldset> Fieldset Groups related elements in a form.
<legend> Legend Defines a caption for a <fieldset> element.
<datalist> Data List Specifies a list of pre-defined options for input controls.
<output> Output Represents the result of a calculation or user action.

Table Tags

Tag Name Description
<table> Table Defines a table.
<caption> Caption Specifies a caption for a table.
<thead> Table Head Groups the header content in a table.
<tbody> Table Body Groups the body content in a table.
<tfoot> Table Foot Groups the footer content in a table.
<tr> Table Row Defines a row in a table.
<th> Table Header Defines a header cell in a table.
<td> Table Data Defines a cell in a table.
<col> Column Specifies column properties for each column within a <colgroup> element.
<colgroup> Column Group Specifies a group of one or more columns in a table for formatting.

Media Tags

Tag Name Description
<img> Image Embeds an image.
<audio> Audio Embeds sound content.
<video> Video Embeds video content.
<source> Source Specifies multiple media resources for <audio> and <video> element.
<track> Track Specifies text tracks for <video> and <audio> elements.
<embed> Embed Embeds external content (e.g., a plugin).
<object> Object Defines an embedded object.
<param> Parameter Defines parameters for an <object> element.
<iframe> Iframe Embeds another HTML page within the current page.
<map> Image Map Defines an image map (with clickable areas).
<area> Area Defines a clickable area inside an image map.
<canvas> Canvas Used to draw graphics, on the fly, via scripting (usually JavaScript).
<svg> SVG Defines vector-based graphics for the web.
<figure> Figure Specifies self-contained content, such as illustrations, diagrams, photos, etc.
<figcaption> Figure Caption Defines a caption for a <figure> element.

Metadata Tags

Tag Name Description
<meta> Meta Defines metadata about an HTML document (e.g., character set, description, keywords, author, and viewport settings).
<link> Link Defines the relationship between the current document and an external resource (most used to link to stylesheets).
<style> Style Contains style information (CSS) for a document.
<base> Base Specifies the base URL for all relative URLs in a document.

Programming Tags

Tag Name Description
<script> Script Embeds executable code or a reference to an external script.
<noscript> No Script Defines alternative content to be displayed if a script is not executed.
